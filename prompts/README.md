# Prompts Especializados para IA - Elias Barao

Esta pasta contem prompts especializados que podem ser usados no inicio de conversas com IAs (ChatGPT, Claude, Gemini, etc.) para que elas tenham contexto completo sobre voce e possam fornecer respostas mais precisas e personalizadas.

## Como Usar

1. Copie o conteudo do prompt desejado
2. Cole no inicio da conversa com a IA
3. Faca sua pergunta ou pedido

## Arquivos Disponiveis

### 1. `master-prompt.md` - Prompt Mestre
**Use quando:** Quiser que a IA tenha TODAS as informacoes sobre voce de uma vez.
**Contem:** Perfil completo, experiencia, projetos, valores, contatos, guidelines.
**Ideal para:** Primeira interacao com uma nova IA ou sessoes de trabalho gerais.

### 2. `dev-prompt.md` - Desenvolvimento
**Use quando:** Precisar de ajuda com codigo, arquitetura, debugging, decisoes tecnicas.
**Contem:** Stack tecnico detalhado, projetos ativos, padroes de codigo, referencias rapidas.
**Ideal para:** Programacao, code review, arquitetura de software, troubleshooting.

### 3. `career-prompt.md` - Carreira e Networking
**Use quando:** Precisar de ajuda com carreira, LinkedIn, propostas, networking.
**Contem:** Perfil profissional, experiencia resumida, diferenciais competitivos, objetivos.
**Ideal para:** Otimizacao de LinkedIn, precificacao, propostas comerciais, networking.

### 4. `projects-prompt.md` - Gestao de Projetos
**Use quando:** Precisar de ajuda com planejamento, organizacao, documentacao.
**Contem:** Status de todos os projetos, metodologias, funcionalidades implementadas/pendentes.
**Ideal para:** Planejamento de sprints, documentacao, arquitetura, priorizacao.

### 5. `personal-prompt.md` - Pessoal e Valores
**Use quando:** A conversa envolver aspectos pessoais, valores, visao de mundo.
**Contem:** Identidade, espiritualidade, trajetoria de vida, valores fundamentais, filosofia.
**Ideal para:** Conversas sobre proposito, identidade, diversidade, espiritualidade.

## Arquivo de Contexto Completo

### `../AI-CONTEXT.md` - Contexto Mestre (Ingles)
Arquivo completo em ingles com TODAS as informacoes. Ideal para IAs que funcionam melhor em ingles ou como referencia completa.

## Dicas

- **Combine prompts:** Para sessoes especificas, combine o master-prompt com um prompt especializado
- **Atualize regularmente:** Mantenha os prompts atualizados com novas experiencias e projetos
- **Adapte conforme necessario:** Sinta-se livre para modificar os prompts para diferentes situacoes
- **Use o contexto do projeto:** IAs como o OpenCode ja tem acesso ao `AI-CONTEXT.md` automaticamente
